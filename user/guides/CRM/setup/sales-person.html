Sales Person

Sales Persons behave exactly like Territories. You can create an organization chart of Sales Persons where each Sales Person’s target can be set individually. Again as in Territory, the target has to be set against Item Group.

Sales Person in Transactions

You can use this Sales Person in Customer and sales transactions like Sales Order, Delivery Note and Sales Invoice. Click here to learn more about how Sales Persons are used in the transactions of Sales Cycle.
